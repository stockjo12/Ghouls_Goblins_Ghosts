# Kaggle Ghouls, Goblins, and Ghosts Competition

Kaggle Competition: [Ghouls, Goblins, and Ghosts] (https://www.kaggle.com/competitions/ghouls-goblins-and-ghosts-boo)

This project is using machine learning methods to try and predict the type of mythical creature we have depending on its characteristics

## Usage
1. Download test and training datasets
2. Download R file
3. Run the code
